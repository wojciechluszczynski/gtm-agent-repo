# gtm-agent-repo
The Claude Code GTM Agent Starter Pack

The exact folder structure, CLAUDE.md, MCP config, and workflow prompts
from the Claude Code GTM Agent Starter Pack.

Full guide and context: [wojciech.io/claude-code-vs-clay](https://wojciech.io/claude-code-vs-clay)

## What's inside
- `CLAUDE.md` — operating manual for Claude Code
- `clients/_template/` — ICP, signals, disqualifiers, examples
- `workflows/` — 3 copy-paste research prompts
- `plugins/gtm-tools/.mcp.json` — MCP server config (Brave, Firecrawl, Exa)

## How to use
1. Fork this repo
2. Copy CLAUDE.md to your Claude Code project root
3. Install MCP tools (see guide)
4. Duplicate `clients/_template/` for each client
5. Run workflows from the `workflows/` folder
